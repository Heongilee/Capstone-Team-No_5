## 주제 :memo:
모바일 플랫폼 기반의 생활 폐기물 품목 측정 기술   
   
모바일 기반의 사용자가 스마트폰 카메라를 이용한 사물 촬영 시 객체 검출을 통해 우리 어플의 폐기물 분류 체계에서 대형 폐기물 중 어떤 항목에 해당 하는지 결과를 보여주고, 나온 결과를 토대로 폐기물 처리 진행을 도와주는 앱.
